# 🌲 Forest Guardian

Forest Guardian is an AI-powered system that detects chainsaw sounds from environmental audio recordings to help identify illegal logging activities.

## Features
- Acoustic monitoring using environmental sound data
- MFCC feature extraction for audio classification
- Random Forest machine learning model
- Interactive web interface built with Streamlit

## Dataset
ESC-50 Environmental Sound Dataset

## Model Performance
Accuracy: 81.25%

## Demo
Upload a forest audio clip and the system will detect whether a chainsaw sound is present.
